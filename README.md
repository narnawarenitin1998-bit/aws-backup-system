# AWS Backup System

Automated backup solution built using Bash scripting, AWS CLI, Amazon S3 and Cron Jobs.

## Features

* Automated file backups
* Compressed archive generation
* Backup logging
* Cron job scheduling
* AWS S3 integration

## Tech Stack

* Linux
* Bash
* AWS CLI
* Amazon S3
* Cron

## Project Workflow

1. Select source directory
2. Create compressed backup
3. Store backup locally
4. Upload backup to Amazon S3
5. Generate log entries

## Future Enhancements

* Email notifications
* Backup retention policy
